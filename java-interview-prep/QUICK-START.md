## 🔥 Critical Rules (Don't Break These)

### **Rule 1: ALWAYS Answer Out Loud**
```
DON'T: Read question mentally in your head.
DO:    SPEAK INTO THE AIR or record yourself.

Reason: Your brain thinks differently when talking vs thinking.
        Interview is talking, so practice talking.
```

### **Rule 2: Never Study >60 Minutes at Once**
```
DON'T: "I have 3 hours free, I'll study 3 hours"
       (Brain focus drops after 50 min anyway)

DO:    45 min focused → 15 min break → repeat.
       Better: 3 × 45min sessions than 1 × 135min.
```

### **Rule 3: Test Before You Read Answer**
```
DON'T: Read question → read answer → think "oh, I knew that"
       (You didn't. You just recognized the answer.)

DO:    Read question → answer from memory → THEN read answer.
       This forces real learning.
```

### **Rule 4: If You Fumble, Flag It Now**
```
DON'T: Ignore weak topics, hope they don't come up
       (They will.)

DO:    During Week 1–3, flag every fumble.
       Week 4–5: Re-study the 10 weak topics.
       Week 6: Confidence check.
```

## 💬 Common Questions Before Starting

### **Q: "I don't have 45 min, can I do 20 min/day?"**
A: No. Below 30 min is inefficient (setup overhead). If you can't do 45 min, pick 3 days/week with 60 min = same total. But consistency matters more than length.

### **Q: "Can I study 2 hours on weekends instead of weekdays?"**
A: No. Spaced repetition requires consistent messaging (daily). One long session = lower retention.

### **Q: "What if I miss a day?"**
A: Don't sweat. Miss max 1 day/week. If you miss 2+ days total, you'll lose momentum. Just start the next day.

### **Q: "Do I need to study from a book / online course in parallel?"**
A: No. This curriculum is complete. Additional sources = confusion. **Depth > breadth.**

### **Q: "How do I know if I'll pass the interview?"**
A: By Week 4, take the full mock interview (section 12). If you score 7–8/10, you're passing. Week 5–6 is confidence building.

## 📞 If You Get Stuck

### **Problem: "I'm reading but nothing sticks"**

→ Solution: You're reading, not testing. Start answering Q&A BEFORE reading the answer. See "Rule 3" above.

### **Problem: "I don't have time for 45 min"**

→ Solution: You have it. You're just prioritizing other things. Be honest. If you can't commit 45 min/day, this may not work. Or find time (wake up earlier, skip 1 episode/day, etc.)

### **Problem: "Some topics are too hard"**

→ Solution: Hard topics need 2 passes. Flag during Week 1–3. Deep-dive in Week 4. This is normal.

### **Problem: "I keep forgetting what I learned"**

→ Solution: You're not doing spaced repetition. Use Anki flashcards (add 5–10 new cards/week). Review 5 min before bed. This fixes forgetting.
